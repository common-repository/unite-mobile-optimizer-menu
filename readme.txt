Unite Mobile Optimizer Menu will create a widget of icons to link to your sites.
== Description ==

Unite Mobile Optimizer Menu will help your to create a sliding icon list dynamically where you can place on your website.

The settings menu allows you to easily choose which icons you want by putting the URL.

== Installation ==

1. Download the plugin then upload 'unite-mobile-optimizer-menu' folder to the '/wp-content/plugins/'  directory
   --OR--
   Download and install the plugin 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go the settings page to put in your URL's
